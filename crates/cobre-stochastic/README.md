# cobre-stochastic

Stochastic process models for the [Cobre](https://github.com/cobre-rs/cobre) power systems ecosystem.

Provides PAR(p) autoregressive models, correlated scenario generation, and Monte Carlo simulation for power system uncertainty modeling.

## Status

**Alpha** — API is functional but not yet stable. See the [main repository](https://github.com/cobre-rs/cobre) for the current release.

See the [main repository](https://github.com/cobre-rs/cobre) for the full roadmap.
