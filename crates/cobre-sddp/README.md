# cobre-sddp

Stochastic Dual Dynamic Programming (SDDP) for the [Cobre](https://github.com/cobre-rs/cobre) power systems ecosystem.

Implements the SDDP algorithm for long-term hydrothermal dispatch and energy planning, with support for CVaR risk measures, multi-cut strategies, and distributed execution via MPI.

## Status

**Alpha** — API is functional but not yet stable. See the [main repository](https://github.com/cobre-rs/cobre) for the current release.

See the [main repository](https://github.com/cobre-rs/cobre) for the full roadmap.
