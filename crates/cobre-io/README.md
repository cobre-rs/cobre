# cobre-io

File parsers and serializers for the [Cobre](https://github.com/cobre-rs/cobre) power systems ecosystem.

Provides interoperability with PSS/E, CSV, Arrow/Parquet, and other power system data formats.

## Status

**Alpha** — API is functional but not yet stable. See the [main repository](https://github.com/cobre-rs/cobre) for the current release.

See the [main repository](https://github.com/cobre-rs/cobre) for the full roadmap.
